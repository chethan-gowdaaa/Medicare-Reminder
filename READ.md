# 🩺 Medicare Reminder

A modern, responsive Progressive Web App (PWA) designed to help patients manage their medication schedules, track medicine inventory, and receive real-time audio reminder alerts.

## 🚀 Features
- 🔐 **Secure Access:** Interactive Login and Registration interface.
- 📊 **Dynamic Dashboard:** View today's scheduled medical alerts at a glance.
- ⏰ **Smart Scheduler:** Easily add new medication times, doses, and targets.
- 📦 **Inventory & Reports:** Track medicine stocks and overall adherence history.
- 🔊 **Audio Reminders:** Automated system alarms that sound exactly when it's time to take a tablet.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Poppins Typography), JavaScript (ES6)
- **Backend Architecture:** Node.js (Express server framework)
- **Database:** Local JSON-based data store configuration (`db.json`)
- **PWA Capabilities:** Service Workers (`sw.js`) and Web Manifest (`manifest.json`) for standalone mobile/desktop installation.
